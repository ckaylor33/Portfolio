    My Skills

    I am a creative problem solver, building fullstack web applications. I use C#, ASP.NET, SQL, Javascript, CSS, HTML and Bootstrap. I have highlighted my key technical skills that I will bring to your business/project below.

    C# Development

    I use C# on the backend to build secure, robust and performant websites. You can see examples of my work implementing C# in my projects section.

    Full Stack Web Development

    I use proven proven design patterns and frameworks to build websites. ASP.NET and MVC fits the bill for most projects. Visit my projects section to see examples.

    Frontend Developer

    I use HTML, CSS and Javascript to build attractive and responsive frontends for websites. To see my frontend work, check out my projects and coding challenges sections.

    Database Development

    Database development is critical to most projects. I am well versed in relational SQL development, from queries to design.

    Frontend Design

    The look and feel are very important elements of any project. I utilise Bootstrap to bring an attractive design to your web projects.

    Build & Deploy

    Using Git and Github to track source code in each projects' repositories. Helps manage workflow and CI/CD when the application is deployed.
