    BugLog

    BugLog is a software defect tracking system. Commonly known as a bug tracker, BugLog is built with ASP.NET, MVC, C# & SQL. A role based security system was implemented to create permission based roles. The roles of Admin, Project Manager, Developer and Submitter restrict what the user can see and do, with company member registration restricted to invite by Admin only. The software allows for users to submit tickets and those tickets are tracked through status from New to Resolved.

    The system was built with C# and .NET over a couple of months.The front-end uses HTML, CSS, Javascript and Bootstrap, while the back-end utilizes MVC and C#. The database was built with SQL (Postgresql) and hosted on Heroku. Please visit the site and try it out.

    CK Blogs

    CK Blogs is my personal blog application built with ASP.NET, MVC, C# & SQL. It has all the features of a standard blog application, including creation of categories and posts, search functionality based on post title/description and tags on posts, comment section on posts for reader interaction, and a security system implemented to create permission based roles. Site visitors can also create an account if they would like to interact/comment on particular posts, but that is the extent of a registered users functionality. Blog/post creation and comment moderation is limited to the Administrator and Moderator roles.

    The blog was built with C# and .NET on and off over a few months, with the plan to improve UI and add features over time - specifically to implement post likes, view count on posts and trending posts display. The front-end uses HTML, CSS, Javascript and Bootstrap, while the back-end utilizes MVC and C#. The database was built with SQL (Postgresql) and hosted on Heroku. Please visit the site and try it out.
